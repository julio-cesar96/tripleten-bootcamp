Exercício 1 – Validador de Senha

Pratique métodos de string, números e lógica básica criando uma função que avalia a força de uma senha.

1
Enunciado

Crie uma função chamada validarSenha(senha) que receba uma string e retorne um objeto com as seguintes informações:

  • Se a senha tem no mínimo 8 caracteres
  • Se contém ao menos um número (dica: use parseInt ou regex)
  • Se contém ao menos uma letra maiúscula
  • Se contém ao menos um caractere especial (!@#$%)
  • A força geral: forte, média ou fraca

2
Dicas de métodos JS

  • .length — tamanho da string
  • .toUpperCase() e .toLowerCase() — alternar case
  • /regex/.test(string) — testar padrão
  • isNaN() / Number() — verificar se é número

function validarSenha(senha) {
  const temTamanho  = senha.length >= 8
  const temNumero   = /[0-9]/.test(senha)
  const temMaiuscula= /[A-Z]/.test(senha)
  const temEspecial = /[!@#$%^&*]/.test(senha)

  const pontos = [temTamanho, temNumero, temMaiuscula, temEspecial]
                  .filter(Boolean).length

  const forca = pontos === 4 ? 'forte'
              : pontos >= 2 ? 'média'
              : 'fraca'

  return { temTamanho, temNumero, temMaiuscula, temEspecial, forca }
}
