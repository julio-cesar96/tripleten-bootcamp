Exercício 2 – Carrinho de Compras

Pratique arrays de objetos, map, filter, reduce e arrow functions construindo a lógica de um mini carrinho.

1
Enunciado

Você tem um array de produtos (objetos). Implemente as seguintes funções:

  • calcularTotal(itens) — soma o preço × quantidade de cada item
  • aplicarDesconto(itens, pct) — retorna o total com % de desconto
  • removerItem(itens, id) — retorna novo array sem o item pelo id
  • itensCaros(itens, limite) — retorna só itens acima do limite de preço

2
Estrutura dos dados

const produtos = [
  { id: 1, nome: 'Teclado Mecânico', preco: 350, qtd: 1 },
  { id: 2, nome: 'Mouse Gamer',       preco: 180, qtd: 1 },
  { id: 3, nome: 'Monitor 24"',       preco: 1200, qtd: 1 },
  { id: 4, nome: 'Headset',           preco: 250, qtd: 1 },
]

3
Template — complete as funções

const calcularTotal = (itens) =>
  itens.reduce((acc, item) => acc + item.preco * item.qtd, 0)

const aplicarDesconto = (itens, pct) =>
  calcularTotal(itens) * (1 - pct / 100)

const removerItem = (itens, id) =>
  itens.filter(item => item.id !== id)

const itensCaros = (itens, limite) =>
  itens.filter(item => item.preco > limite)
